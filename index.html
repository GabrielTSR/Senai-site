FPOO

Java

Como um dos conteúdos propostos no curso de "Desenvolvimento de Sistemas", Java é um exemplo de linguagem de prorgamação orientada a objetos, utilizando-a, é possível criarmos aplicações com propósitos diversos, mas que no geral, estão ligados à automatização de tarefas (criar aplicativos desktops, manipular o back-end).

Eclipse IDE

Eclipse é uma IDE para desenvolvimento Java, porém suporta várias outras linguagens a partir de plugins como C/C++, PHP, ColdFusion, Python, Scala e Kotlin. Assim como em todas as profissões, necessitamos de ferramentas para alcançar o devido objetivo, com programação não é diferente. Utilizamos recursos como a "Eclipse IDE", que nos permite programar de maneira mais ágil e eficiente, agilizando o processo no geral.

LIMA

HTML

Linguagem de marcação, servindo como base para toda a estrutura de um site (páginas que serão reconhecidas e interpretadas por um navegador WEB).

CSS

O que em português seria "Folha de Estilo em Cascatas", é responsável pela estilização do projeto, se fôssemos comparar o front-end com o corpo humano, HTML seria o esqueleto, CSS a aparência, e Java Script as nossas ações.

SOP

CONHECENDO E INSTALANDO SISTEMAS OPERACIONAIS

Precisamos garantir que as nossas aplicações funcionem corretamente na maior gama de dispositivos e sistemas operacionais, e por isso, é importante que os alunos saibam lidar com essa diversidade tecnológica. Além de SOP ser uma boa disciplina para tornar essa inclusão possível, é uma boa aliada na busca de descobrir ferramentas que podem aumentar nossa eficiência.

Excel

Como um dos recursos presentes no Sistema Operacional "Windows 10", conseguimos utiliza-lo para lidar com diversas planilhas, servindo como uma forma eficaz para automatizar cálculos, criar gráficos, e etc.

Instalando o Windows 10

Uma demonstração dos conteúdos aprendidos durante a aula. Nesse vídeo, você confere o processo de instalação do Windows 10 em uma máquina virtual.

Virtal Box

Com o auxílio desta ferramenta, é possível instalarmos diversos sistemas operacionais (sem a necessidade de particionarmos efeticamente um disco rígido), tornando possível testar a compatibilidade dos nossos programas com sistemas operacionais diferentes.

HARE

Sistemas de numeração

Desenvolvedores farão aplicações para dispositivos eletrônicos, esses dispositivos tendem a utilizar sistemas de numeração diferentes do que estamos acostumados (decimal), tornando crucial o conhecimento sobre o assunto.

Endereçamento IP

Para que haja conexão entre dispositivos, é necessário que o desenvolvedor tenha ciência do funcionamento dos endereços das máquinas, por exemplo, tornando possível a conexão entre uma aplicação e um servidor, que também poderá se conectar com diversos dispositivos.

Funcionamento do computador

Sendo o nosso foco aplicações para dispositivos, como computadores, notebooks, e celulares, é necessário que saibamos como eles funcionam, para assim, lidar corretamente com diversas situações relacionadas ao hardware.
